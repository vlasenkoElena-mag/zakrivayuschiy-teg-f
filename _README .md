# Закрывающий тег

Учебный проект с адаптивной вёрсткой в стиле ретро-страницы. Внутри собраны карточки с изображениями, интерактивные кнопки лайка, модальное окно и аккуратная семантическая структура страницы.

## Репозиторий

(https://github.com/vlasenkoElena-mag/zakrivayuschiy-teg-f)

## Что есть в проекте

- адаптивная главная страница с шапкой, списком карточек и кнопкой сохранения;
- интерактивные карточки с лайками, работающими через JavaScript;
- модальное окно на `dialog`;
- фиксированный фон с декоративными слоями;
- CSS Grid и Flexbox для построения сетки;
- изображения карточек с `loading="lazy"`.

## Стек

- HTML5;
- CSS3;
- JavaScript;
- SVG и растровые изображения.

## Структура

- `index.html` — основная разметка страницы;
- `fonts/fonts.css` — подключение шрифтов;
- `styles/variables.css` — CSS-переменные;
- `styles/globals.css` — базовый сброс и утилиты;
- `styles/style.css` — основные стили страницы;
- `styles/animations.css` — анимации;
- `scripts/like.js` — логика лайков;
- `images/` — растровые изображения;
- `svg/` — SVG-иконки.

## Как открыть

1. Откройте `index.html` в браузере.
